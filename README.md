Student Excel Report Generator
The Student Excel Report Generator is a Python-based application developed to manage and analyze student academic records stored in an Excel workbook. The project demonstrates the use of Python programming concepts such as functions, loops, dictionaries, lists, exception handling, and Excel file operations using the openpyxl library.
The application reads student records from an Excel file, performs statistical analysis on the data, allows searching and filtering of students based on their marks, and automatically generates a summary worksheet containing key performance metrics.
This project was developed as part of a Python Automation Internship to gain practical experience in data processing and Excel automation using Python.

Features
•	Read student data from Excel
•	Find highest and lowest scoring students
•	Calculate class average
•	Count pass and fail students
•	Search students by name
•	Filter students above 75 marks
•	Filter students below 35 marks
•	Generate a Summary worksheet in Excel
•	Handle file and data-related exceptions

Technologies Used
•	Python
•	openpyxl
•	Excel (.xlsx)

Files
•	student_report.py – Source code
•	student.xlsx – Input and output Excel file
•	README.md – Project documentation

Output
The program creates a Summary worksheet containing:
•	Highest Scorer
•	Lowest Scorer
•	Average Marks
•	Pass Count
•	Fail Count

Conclusion
The Student Excel Report Generator is a simple yet effective automation project that demonstrates how Python can be used to manage and analyze Excel-based student records. It combines file handling, data analysis, report generation, and error handling into a practical real-world application, making it an excellent beginner-level automation project.
